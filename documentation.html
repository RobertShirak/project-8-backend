Routes for Sculpt Fitness API

ROOT URL is https://sculpt-fitness.herokuapp.com/


/cardio/

GET to view all cardio activity

/cardio/specific/:id

GET with a specific ObjectID to retrieve 1 cardio activity

/cardio/add

POST to add a new cardio activity

/cardio/update/:id

PUT to modify an existing cardio activity

/cardio/delete/:id

DELETE to remove a cardio activity
