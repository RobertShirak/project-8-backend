Routes for Sculpt Fitness API

ROOT URL is https://sculpt-fitness.herokuapp.com/


/cardio/

GET to view all cardio activity

/cardio/add

POST to add a new cardio activity

/cardio/update/:id

PUT to modify an existing cardio activity

/cardio/delete/:id

DELETE to remove a cardio activity


/lifting/

GET to view all lifting activity

/lifting/add

POST to add a new lifting activity

/lifting/update/:id

PUT to modify an existing lifting activity

/lifting/delete/:id

DELETE to remove a lifting activity
